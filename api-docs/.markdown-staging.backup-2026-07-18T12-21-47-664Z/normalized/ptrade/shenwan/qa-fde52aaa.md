<a id="关于研究"></a>

## 关于研究

<a id="研究环境资源"></a>

### 研究环境资源

目前单个的客户研究环境资源的上限情况，由所在券商决定。

<a id="文件传输限制"></a>

### 文件传输限制

单个文件手动上传下载目前限制单个文件数据大小不超过50M。

<a id="关于定时上传功能"></a>

### 关于定时上传功能

定时上传功能可以实现研究环境自动从本地路径读取文件，但上传次数是有限制的（具体由券商配置决定），因此仍不能支持高频的策略中调用本地数据场景。

定时上传功能目前限制单个文件数据大小不超过50M。

<a id="上传文件夹"></a>

### 上传文件夹

暂时不支持文件夹上传，文件需要逐个上传。
