## 回测支持业务类型

目前所支持的业务类型:

1.  普通股票买卖(单位：股)。

2.  可转债买卖(单位：张，T+0)。

3.  融资融券担保品买卖(单位：股)。

4.  期货投机类型交易(单位：手，T+0)。

5.  LOF 基金买卖(单位：股)。

6.  ETF 基金买卖(单位：股)。
