## 新建交易

交易界面点击新增按键进行新增交易操作，策略方案中的对象为所有策略列表中的策略，给本次交易设定名称并点击确定后系统就开始运行交易了。

![](http://101.71.132.53:9091/qthelp/static/images/help/creat_trade.png)

交易开始运行后，可以实时看到总资产和可用资金情况，同时可以在交易列表查询[交易状态](help-engine-e187e04d.md#trade_status)。

![](http://101.71.132.53:9091/qthelp/static/images/help/account_info.png)

交易开始运行后，可以点击交易详情，查看策略评价指标、交易明细、持仓明细、交易日志。

![](http://101.71.132.53:9091/qthelp/static/images/help/trade_info.png)
