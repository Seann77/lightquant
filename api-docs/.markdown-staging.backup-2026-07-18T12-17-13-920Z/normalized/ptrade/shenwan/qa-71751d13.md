# 常见问题

<a id="关于量化平台部署"></a>

## 关于量化平台部署

<a id="关于量化环境"></a>

### 关于量化环境

PTrade客户端为本地运行，但量化模块为云端部署方式，客户端通过http或https方式链接云服务器。

<a id="关于策略安全"></a>

### 关于策略安全

客户策略部署在券商服务器，策略文件加密存储。利用Docker容器技术，隔离多账号运行空间，有效分配各用户账号运行环境，保证策略运行环境与资源使用的独立性。

<a id="mac是否支持"></a>

### Mac是否支持

Mac目前不支持，只能在Windows环境下。

<a id="关于付费"></a>

### 关于付费

1.  云纪网络提供的仿真环境为付费使用，包括所有数据、回测资源、模拟交易资源、答疑服务等。
2.  不同版本对应不同功能和费用，详见链接：`https://www.hs.net/openplat-front/service/serviceDetail/307`。
3.  券商部署环境以券商发布的使用条件为准。

<a id="关于程序语言"></a>

### 关于程序语言

PTrade量化目前仅支持Python语言编程。

<a id="关于python版本"></a>

### 关于Python版本

目前支持 Python 3.11.4。
