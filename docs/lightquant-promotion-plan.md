# LightQuant 推广执行计划

更新时间：2026-06-22

## 1. 推广目标

LightQuant 当前适合用“AI 量化策略代码助手”定位进入市场，而不是“荐股/收益工具”。核心卖点应稳定表达为：

- 用自然语言生成或修改 PTrade、聚宽 JoinQuant、QMT 策略代码。
- 把聚宽、PTrade、QMT 等平台策略代码迁移到目标平台，并提示 API 差异与人工复核点。
- 把难读的量化策略代码解析成中文报告，帮助用户理解交易逻辑、参数、风险点和优化方向。
- 所有内容仅用于学习、研究和回测参考，不承诺收益，不提供投资建议。

90 天推广目标建议：

| 阶段 | 时间 | 核心目标 | 建议结果 |
| --- | --- | --- | --- |
| 冷启动准备 | 第 0-1 周 | 完成 SEO 基础、账号矩阵、素材库、合规话术 | 8-12 篇基础内容，3 个可领取资料，完整埋点 |
| 搜索收录与内容启动 | 第 2-4 周 | 抢长尾关键词，验证首批转化渠道 | 20+ SEO 页面/文章，50-150 注册，20-60 激活用户 |
| 社区渗透与案例沉淀 | 第 5-8 周 | 在量化社区形成“工具型认知” | 8-12 个匿名案例，2 场直播/答疑，100-300 激活用户 |
| 转化放大 | 第 9-12 周 | 优化注册、首任务、充值闭环 | 300-800 注册，30-80 付费/充值用户，形成复盘报表 |

核心北极星指标：每周完成至少 1 次有效 AI 任务的用户数。

辅助指标：自然搜索点击、注册率、首任务完成率、二次任务率、社群进群率、充值转化率、每渠道获客成本。

## 2. 用户分层

| 用户 | 痛点 | 主推功能 | 内容角度 |
| --- | --- | --- | --- |
| 量化新手 | 有策略想法但不会写平台代码 | 策略生成/修改 | “一句话生成双均线策略骨架” |
| Python/量化学习者 | 能看代码但不懂策略结构 | 代码解析 | “把策略代码拆成交易逻辑、参数和风险点” |
| 平台迁移用户 | 聚宽、PTrade、QMT API 差异大 | 平台代码转换 | “JoinQuant 转 PTrade 的常见坑” |
| 券商量化终端用户 | 需要快速改策略、查错误 | 策略修改/代码解析 | “调仓函数、持仓字段、行情接口怎么复核” |
| 教学/社群运营者 | 需要案例和工具提升教学效率 | 批量案例演示 | “用 AI 做策略教学助教” |

## 3. SEO 计划

### 3.1 当前站点基础

已具备：

- `public/robots.txt` 已允许抓取，并声明 sitemap。
- `public/sitemap.xml` 已包含首页、策略生成、平台转换、代码解析、更多页。
- 首页已有 title、description 和核心功能入口。
- 已有百度验证文件。

需要补齐：

- sitemap 目前只有少量产品入口页，必须增加长尾内容页。
- 当前功能页使用 query 参数，例如 `/chat?mode=strategy`，建议补充可索引的静态落地页，例如 `/strategy-generator`、`/platform-converter`、`/code-analysis`。
- 为每个核心页面增加 canonical、Open Graph、结构化 FAQ、面包屑。
- 增加百度搜索资源平台、Google Search Console、Bing Webmaster Tools 提交和数据监测。
- 站内需要“案例页/教程页/模板页/词典页”，不能只依赖首页转化。

### 4.2 微信视频号/抖音

注意：

- 不展示收益率神话。
- 不说“稳赚”“高胜率”“自动赚钱”。

每周：

- 发布 2 个匿名案例：代码解析、平台转换、策略生成各轮换。
- 发起 1 次“代码解析日”活动。
- 在聚宽、BigQuant、vn.py 等社区各贡献 1-2 篇高质量内容或回答。
- 做 1 次直播/语音答疑：主题围绕策略迁移或代码解析。

重点：

- 收集用户原话，但只使用“节省时间、理解代码、迁移更清楚”类反馈。
- 不展示收益截图或收益承诺。
- 将高频问题整理成 FAQ。

### 第 9-12 周：转化优化

每周：

- 对高转化主题加写系列文章。
- 对高播放视频做二剪和图文复用。
- 做小额平台测试投放，优先测试 B站、小红书、知乎内容加热。
- 尝试与量化课程博主、Python 教学博主、券商量化内容博主合作。

转化实验：

- 版本 A：注册送积分。
- 版本 B：领取平台差异表。
- 版本 C：免费解析一次脱敏代码。
- 对比注册率、首任务率、充值率。

## 10. 合规红线

禁止：

- 承诺收益、保本、无风险、高胜率、稳赚。
- 推荐具体股票、给出买卖点、指导实盘交易。
- 展示夸张收益截图诱导购买。
- 用“内幕”“暴利”“躺赚”“自动赚钱”等表达。
- 收费提供个股投资分析、预测或建议。

推荐固定声明：

> LightQuant 输出内容仅用于量化策略学习、研究和回测参考，不构成任何投资建议或实盘交易依据。量化交易存在风险，策略代码在实盘前必须由用户自行理解、验证和复核。

营销话术替代：

| 不建议 | 建议 |
| --- | --- |
| AI 帮你赚钱 | AI 帮你生成策略代码初稿 |
| 高胜率策略 | 便于回测验证的策略框架 |
| 稳定收益 | 结构清晰、可复核的代码结果 |
| 自动荐股 | 代码解析、平台转换、研究辅助 |
| 跟着买 | 用于学习、研究和回测参考 |
